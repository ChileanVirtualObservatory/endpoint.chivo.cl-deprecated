Welcome to ChiVO Web Service

Alma data requests

We implemented TAP, SCS, SIA , SSA services, to request any of them just send via GET/POST the query with parameters to dachs.lirae.cl/alma/{service} (service can be tap,scs,sia,ssa)

To get the access url to other catalogs

Go to dachs.lirae.cl/external/registry or dachs.lirae.cl/external/{service}

We use Json responses
